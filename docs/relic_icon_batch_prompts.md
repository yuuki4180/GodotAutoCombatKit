# Relic Icon Batch Prompts

Use this common prefix for every 3x3 batch:

```text
Create a 3x3 contact sheet of cute chibi fantasy relic icons for a cozy 2D mobile auto-battler.
Each cell contains exactly one isolated object icon, centered, readable at small mobile UI size, with a simple rounded silhouette.
Style direction: cute magical dungeon items, soft navy and powder-blue shadows, white highlights, small red or violet accents, gentle spooky charm rather than horror.
Rendering: flat-to-soft 2D illustration, thick clean dark outline, rounded edges, simple shapes, low detail, toy-like but not glossy, matching a chibi sailor-style player character.
Use a consistent front-facing or slight 3/4 view, consistent icon scale, no frame, no pedestal, no background scene.
Transparent background or plain very light neutral background.

Avoid AI-generated look: no realistic metal rendering, no 3D render, no hyper-detailed fantasy concept art, no complex filigree, no dramatic smoke, no heavy bloom, no particle storm, no photorealism, no gritty texture, no text, no labels, no numbers, no watermark.
Icons should feel like small cute collectibles from the same game as a round chibi player with pale blue hair, navy outfit, simple face, thick black outline, and soft pastel shading.
Keep every object compact, readable, and friendly, even when the relic theme is cursed, blood, demon, poison, or skull.

Objects, left to right, top to bottom:
```
