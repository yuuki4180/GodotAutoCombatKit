# Asset Policy

Godot Auto Combat Kit includes prototype art so contributors can run and inspect the project without downloading separate asset packs.

## Repository Assets

Assets committed under `assets/` are intended for use with this open-source reference project. They are included to support:

- local development;
- screenshots and documentation;
- prototype gameplay verification;
- pull request review.

Do not assume assets from this repository can be reused in unrelated commercial projects unless their source and license are explicitly documented.

## Adding Assets

When adding or replacing assets:

- include only assets you have the right to redistribute;
- avoid copied art, logos, names, or character designs from commercial games;
- prefer original, generated, or permissively licensed prototype assets;
- document source and license notes in the pull request;
- keep file names descriptive and stable.

## Generated Assets

Generated assets are acceptable when they are safe to redistribute and do not intentionally imitate protected characters, brands, logos, or game-specific art direction.

## Review Checklist

Before merging asset changes, maintainers should check:

- the asset is needed for the runnable prototype or docs;
- the source is clear enough for future maintainers;
- the file size is reasonable;
- the asset does not create brand or trademark confusion;
- the README, NOTICE, or relevant docs are updated if licensing context changes.
