# Demo Media Guide

README media is important because reviewers and contributors should be able to understand the playable prototype quickly.

## Target Capture

Capture a short portrait clip or screenshot showing:

- the 720 x 1280 mobile viewport;
- drag or touch-style movement;
- auto attacks targeting nearby enemies;
- pickups on the ground;
- a level-up, chest, altar, elite, or guardian moment if possible.

## Recommended Formats

- Screenshot: PNG, under 2 MB when possible.
- Short clip: GIF or MP4, 8 to 12 seconds.
- File location: `docs/media/`.

## README Placement

Place the demo near the top of `README.md`, after the opening description and before "Why This Exists".

Use alt text that describes the scene, for example:

```md
![Godot Auto Combat Kit gameplay: portrait arena with auto attacks, enemies, pickups, and upgrade flow](docs/media/gameplay-preview.gif)
```

## Capture Notes

Do not include desktop notifications, private file paths, Apple signing details, or unrelated browser/application windows in demo media.
