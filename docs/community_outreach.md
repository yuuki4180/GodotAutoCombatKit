## Godot Forum / Reddit Draft

```text
I published Godot Auto Combat Kit, an open-source Godot 4 reference project for mobile auto-combat game prototypes.

The goal is not to ship a finished game yet. The goal is to make a compact, inspectable project showing how phone-first action systems fit together:

- portrait 720 x 1280 layout
- touch/mouse drag movement and WASD testing controls
- nearest-target auto attacks
- wave pressure, elites, and guardian encounter
- XP, pickups, level-up choices, weapons, charms, and relic-style modifiers
- public-safe iOS export notes
- contribution, security, roadmap, and testing docs

I would appreciate feedback on Godot structure, mobile readability, and good first issues for contributors.

Repo: https://github.com/yuuki4180/GodotAutoCombatKit
```
