## Assets

The `assets/` directory contains prototype game art, icons, effects, and generated visual assets used by this project. These assets are included so contributors can run and inspect the prototype without needing external downloads.

If you replace, add, or regenerate assets, include source notes in the pull request and avoid adding third-party assets unless their license allows redistribution in this repository.

## Trademarks And References

Dadasivive is an independent prototype. Names of other games or tools in documentation are used only as design references.
