# Support

Dadasivive is an early-stage open-source prototype maintained in public.

## Questions

For general questions, open a GitHub issue using the feature request template and describe what you are trying to understand or build.

## Bugs

For gameplay, Godot, or export bugs, open a bug report with:

- Godot version;
- platform and device;
- reproduction steps;
- expected behavior;
- actual behavior;
- screenshots, video, or logs when helpful.

## Security

Do not open public issues for security-sensitive reports. Follow [SECURITY.md](SECURITY.md).
